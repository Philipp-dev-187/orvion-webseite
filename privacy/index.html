3. Downloads and updates

The Orvion installer is distributed through GitHub Releases. If you download Orvion, GitHub may process technical download and access data according to GitHub's own privacy terms.

The app uses Sparkle for updates. Sparkle checks the public update feed at https://www.getorvion.app/appcast.xml and downloads signed, notarized update files from GitHub Releases. Update checks may involve technical request data such as IP address, time of request, app version, and system compatibility information needed to determine whether an update is available.

4. Payments and licenses

Paid licenses are sold through Lemon Squeezy. If you buy a license, Lemon Squeezy handles checkout, payment processing, invoices, tax handling, fraud prevention, and license key delivery as an external payment and licensing provider.

Inside the app, license activation and validation use Lemon Squeezy's license API. The app may send the license key and a device or activation identifier needed to activate, validate, or deactivate a license. The app stores license state locally, including in the macOS Keychain where appropriate.

5. App data and permissions

Orvion is designed as a local macOS utility. Feature settings, shortcut configuration, layout information, local preferences, trial status, and license state are stored on your Mac. App features may process local information such as window state, selected shortcuts, playback state, clipboard history, system stats, focus timer state, and notch layout settings.

Depending on the features you enable, Orvion may request macOS permissions for Accessibility, Input Monitoring, Camera, Calendars, Reminders, Automation / Apple Events, network access, and user-selected files. These permissions are used to provide the requested local app features, such as window management, input remapping, camera mirror, calendar/reminder widgets, media control, license checks, and updates.

Clipboard history and similar utility data are intended to remain local unless you manually copy, export, paste, or otherwise share that content yourself.

6. Cookies, analytics, and tracking

The current website code does not set optional tracking cookies and does not include analytics or advertising scripts. The language switcher may store your selected language locally in the browser so the site opens in the same language next time.

If analytics, advertising, newsletter tools, or additional third-party embeds are added later, this policy will be updated before or at the time those tools are introduced.
