FAQ

A few things worth knowing.

Can I try Orvion before buying?

Yes. The first launch starts a 7-day trial with all features unlocked.

Can I disable features I do not use?

Yes. Orvion is modular, so you can keep your setup focused and turn off tools you do not need.

Why does Orvion ask for macOS permissions?

Only specific features need permissions: Accessibility for window management, Input Monitoring for remapping, Camera for mirror, Calendars and Reminders for widgets, and Automation for supported app control.

How are updates installed?

Orvion uses Sparkle to deliver signed and notarized app updates.
