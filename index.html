FAQ

Good things to know before installing.

Do I need to pay before trying Orvion?

No. The app starts with a 7-day full trial on first launch.

Can I disable features I do not use?

Yes. Orvion is modular, so you can keep only the tools that fit your setup.

Why does Orvion ask for macOS permissions?

Some features need system permissions: Accessibility for window management, Input Monitoring for remapping, Camera for mirror, Calendars and Reminders for widgets, and Automation for supported app control.

How are updates installed?

Updates are delivered with Sparkle using signed and notarized release files.
